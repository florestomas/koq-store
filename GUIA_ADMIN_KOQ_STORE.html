Guía para Administradores

Esta guía es para el encargado o administrador del sistema. Además de todo lo que puede hacer un usuario normal (vender, recibir, consultar stock), como administrador tiene acceso a herramientas especiales: puede crear productos, editar stock, modificar precios, eliminar registros, agregar colores y talles, y gestionar categorías. Esta guía explica solo las funciones de administrador, asumiendo que ya leyó la guía general.

1. El modo ADMIN — Cómo activar las herramientas de admin

Para ver los botones y controles de administrador, primero tiene que activar el modo ADMIN.

Paso 1: Mire la barra violeta del menú, en el costado izquierdo.

Paso 2: Abajo de todo, cerca de su nombre, hay una palanquita que dice ADMIN.

Paso 3: Toque esa palanquita para que se ponga de color. Cuando está activa, aparecen botones nuevos en varias pantallas.

Paso 4: Toque de nuevo para desactivarlo si no quiere ver los botones de admin.

Consejo: Cuando el modo ADMIN está activo, también puede filtrar el catálogo por local y ver el stock de cada sucursal por separado.

2. Editar un producto — Cambiar nombre, foto, colores, talles, precios y stock

Como admin, puede modificar cualquier producto del sistema. Desde cambiarle el nombre hasta ajustar el stock unidad por unidad.

Cómo abrir el editor de un producto

Paso 1: Vaya al Catálogo.

Paso 2: Busque el producto que quiere modificar.

Paso 3: Toque el icono de lápiz (✏) que está en la esquina de la tarjeta del producto. Se abre una ventana grande con varias pestañas.

Las pestañas del editor

Datos básicos

Acá puede cambiar el nombre del producto y la categoría. Toque GUARDAR cuando termine.

Stock

Acá puede ver y modificar el stock de cada local. Hay una tabla por cada sucursal, con colores en las filas y talles en las columnas. Toque cualquier número y escríbalo de nuevo para cambiarlo.

Variantes

Acá puede agregar o sacar colores y talles. También puede cambiar la foto del producto.

Precios

Acá puede cambiar los precios de cada talle. Toque el precio, escríbalo de nuevo y presione Enter para guardar.

Editar stock — explicación detallada

En la pestaña Stock va a ver algo muy parecido a la grilla del catálogo, pero con números que puede tocar y cambiar. Hay un bloque por cada sucursal (Taller, Luz, Santiago, Avellaneda).

Para cambiar el stock de una variante: Toque el número que está en el casillero de ese color × talle. Escríba el número nuevo y presione Enter (o toque afuera). El cambio se guarda automáticamente.

Para cambiar el stock total de un local: Arriba de cada tabla de sucursal hay un número que dice "Stock total". Tóquelo, escriba el nuevo total y el sistema va a distribuir el stock entre todas las variantes.

Para cambiar el stock mínimo: Debajo de cada número de stock hay un número chiquito con borde punteado. Ese es el "mínimo" (la cantidad que quiere tener siempre). Puede cambiarlo igual que el stock normal.

¿Para qué sirve el stock mínimo? El sistema le avisa con una alerta cuando el stock está por debajo de ese número. Por ejemplo, si pone mínimo 5 en una remera negra talle 2, cuando queden 4 o menos va a aparecer una alerta.

Editar colores y talles — explicación detallada

En la pestaña Variantes puede:

  • Agregar un color nuevo: Escriba el nombre del color y toque AGREGAR. El sistema crea el color y agrega ese color a todos los talles del producto (con stock 0). Si el color ya existe en la base de datos, lo reusa.
  • Eliminar un color: Toque la ✕ al lado del color. Cuidado: no se puede eliminar si ese color tiene stock en algún local.
  • Agregar un talle nuevo: Escriba el número de talle y toque AGREGAR. Se agrega ese talle a todos los colores del producto.
  • Eliminar un talle: Toque la ✕ al lado del talle. No se puede eliminar si tiene stock.
Precaución: Cuando agrega un color o talle nuevo, el stock de esas variantes se crea en 0. Después tiene que ir a la pestaña "Stock" para poner las cantidades, o hacer un Ingreso.

Cambiar la foto del producto

En la pestaña Datos básicos o la pestaña Variantes, ve la foto actual del producto. Para cambiarla, toque la imagen o el área punteada y seleccione un archivo de su computadora. La foto se sube automáticamente.

3. Editar stock manualmente — Cambiar cantidades desde el catálogo

Además del editor, hay una forma rápida de cambiar stock directamente desde el catálogo, sin abrir la ventana de edición.

Paso 1: En el Catálogo, active el modo ADMIN.

Paso 2: En el filtro de ubicación (al lado del filtro de stock), elija un local. Ahora las tarjetas muestran el stock de ese local solamente.

Paso 3: Toque la tarjeta del producto para expandirla y ver la grilla de stock.

Paso 4: Toque el icono de lápiz (✏) para abrir el editor, o vaya directo al historial si necesita hacer cambios masivos.

Para cambios rápidos de stock, lo más práctico es usar la pestaña Stock del editor de producto, donde puede modificar número por número.

4. Historial — Editar y eliminar ventas, traslados e ingresos

Como administrador, en el Historial puede modificar o borrar cualquier evento. Esto es útil para corregir errores, pero también es peligroso si no se usa con cuidado.

Cómo editar un evento

Paso 1: Vaya al Historial.

Paso 2: Busque el evento que quiere modificar (venta, traslado o ingreso).

Paso 3: Toque la línea del evento para expandirlo y ver los detalles.

Paso 4: Toque el botón EDITAR (solo visible con modo ADMIN activo).

Paso 5: El sistema lo lleva a la pantalla correspondiente (Ventas, Trasladar Stock o Ingreso) con todos los datos cargados. Haga los cambios y confirme.

Cómo eliminar un evento

Paso 1: Expanda el evento en el Historial.

Paso 2: Toque el botón ELIMINAR.

Paso 3: El sistema le pregunta "¿Está seguro?". Toque "Aceptar".

IMPORTANTE — Qué pasa cuando elimina:
  • Si elimina una venta: El stock de los productos vuelve al inventario como si la venta nunca hubiera pasado.
  • Si elimina un traslado pendiente: El stock se queda donde estaba, el traslado desaparece.
  • Si elimina un traslado confirmado: El stock vuelve al local de origen y se descuenta del destino.
  • Si elimina un ingreso: El stock se descuenta del inventario.

Cómo hacer una devolución de venta

Cuando un cliente devuelve una prenda, no tiene que eliminar la venta. Use el botón HACER DEVOLUCIÓN:

Paso 1: Expanda la venta en el Historial.

Paso 2: Toque HACER DEVOLUCIÓN.

Paso 3: Confirme la pregunta.

Paso 4: La venta queda marcada como "DEVUELTA" en rojo y el stock vuelve al inventario.

Diferencia entre DEVOLUCIÓN y ELIMINAR: La devolución conserva el registro de la venta (marcada como devuelta) para que quede en el historial. Eliminar borra la venta para siempre. Siempre que pueda, use "HACER DEVOLUCIÓN" en vez de eliminar.

5. Crear producto — Más detalles para el admin

Como administrador, cuando crea un producto tiene control total sobre los datos iniciales. La pantalla es la misma que para los usuarios del taller, pero usted puede además decidir los precios finales y el stock inicial en todas las sucursales (no solo en Taller).

Campos importantes al crear

CampoDetalle para el admin
Precios Puede poner distintos precios por talle. Por ejemplo: talle 2 a $5000, talle 4 a $6000. Use el botón ↻ para copiar un precio a todos los talles si son iguales.
Stock inicial En la tabla "STOCK EN TALLER POR VARIANTE", cada variante (color × talle) tiene dos números: "Stock" (cuántas unidades hay) y "Mín" (stock mínimo para alertas). Ponga un mínimo razonable para cada producto.
Foto Subir una foto ayuda a identificar el producto en las pantallas de venta y recepción. Si no sube ninguna, se muestra un icono genérico.
¿Qué stock mínimo poner? Depende de la velocidad de venta. Una remera básica que se vende mucho puede tener mínimo 10. Un producto de temporada que se vende poco puede tener mínimo 2.

6. Gestionar categorías — Agregar, renombrar y eliminar categorías

Las categorías son los grupos de productos: Remeras, Buzos, Camisas, etc. Como admin, puede administrar estas categorías desde el Catálogo.

Cómo gestionar categorías

Paso 1: Vaya al Catálogo y active el modo ADMIN.

Paso 2: Arriba a la derecha, al lado del título "CATÁLOGO", va a ver un icono de engranaje (⚙). Tóquelo.

Paso 3: Se abre una ventana donde puede ver todas las categorías existentes.

Paso 4: Para agregar una categoría nueva: escriba el nombre y toque AGREGAR.

Paso 5: Para renombrar: toque el lápiz ✏ al lado de la categoría, cambie el nombre y confirme.

Paso 6: Para eliminar: toque el icono de papelera 🗑 al lado de la categoría.

Precaución: Si elimina una categoría que tiene productos adentro, esos productos se quedan sin categoría. Es mejor renombrarla que eliminarla.

7. Alertas — Filtrar por local

Como admin, en la página de Alertas puede ver las alertas de cada sucursal por separado. Esto le ayuda a saber qué falta reponer en cada local.

Paso 1: Vaya a ALERTAS.

Paso 2: Abajo de las tarjetas de resumen, va a ver botones con los nombres de las sucursales: "Todas", "Taller", "Luz", "Santiago", "Avellaneda".

Paso 3: Toque una sucursal para ver solo las alertas de ese local.

Paso 4: Toque "Todas" para volver a ver todo junto.

También puede usar el botón EXPORTAR / IMPRIMIR para sacar un listado impreso de las alertas y dárselo al encargado del taller para que sepa qué producir.

8. Eliminar productos y registros — Precauciones importantes

Como administrador, tiene el poder de eliminar cosas del sistema. Esto es útil para limpiar datos incorrectos, pero hay que hacerlo con cuidado.

Eliminar un producto completo (modelo)

Paso 1: En el Catálogo, expanda la tarjeta del producto que quiere eliminar.

Paso 2: Abajo de todo, va a ver un botón rojo que dice ELIMINAR PRODUCTO.

Paso 3: Toque ese botón. El sistema le pregunta "¿Eliminar este producto definitivamente?".

Paso 4: Si toca "Aceptar", se borra todo: el modelo, los colores asociados, los productos de todos los talles y colores, y todos los registros de stock.

MUY IMPORTANTE:
  • Eliminar un producto borra para siempre su historial de ventas, traslados e ingresos (los registros en esas tablas también se eliminan).
  • Si el producto tiene stock en algún local, ese stock desaparece.
  • Los colores que solo usaba ese producto también se eliminan de la base de datos.
  • No se puede deshacer. Piense dos veces antes de hacerlo.

Regla general: cuándo eliminar y cuándo no

Situación¿Eliminar?Alternativa
Un producto ya no se fabrica másNoMejor desactivarlo (no hay botón todavía, pero puede dejar el stock en 0)
Una venta se hizo por errorNoUse HACER DEVOLUCIÓN para que quede registrada
Un traslado se canceló antes de enviarSí, si está pendiente—
Un producto de prueba que nunca se usóSí—
Un color o talle que ya no se usaEn el editor de productoSolo si no tiene stock

Reglas de oro para el administrador

1. Antes de eliminar, pregúntese: "¿Puedo editarlo en vez de borrarlo?" Si la respuesta es sí, edite.
2. Cuando edite stock, verifique dos veces el número que escribió. Un error puede dejar stock negativo o cantidades incorrectas.
3. Use "HACER DEVOLUCIÓN" para ventas, no "ELIMINAR". Así queda registro de la devolución.
4. Revise las alertas seguido. Es la mejor herramienta para saber qué producir.
5. Si no está seguro de algo, consulte con otro administrador antes de hacer un cambio irreversible.
